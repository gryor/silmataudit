Oireet
------

- Kipu
- Punoitus
- Valonarkuus
- Näkö heikentynyt


Löydökset
---------

- Sidekalvo

 - Punoitus sarveiskalvon reunaa kiertäen

- Sarveiskalvo

 - Tartunta
 - Pinnallinen vaurio
 - Väliaineen sameus
 - Sisäkalvopoimut (Descementinkalvo)
 - Sisäpinnan presipitaatit (herpes?)

- Etukammio

 - Solut
 - Valotie
 - Alaosan märkäkertymä (hypopyon)

- Kammiokulma

  - Tumankappale **kaihileikkauksen jälkeen**
